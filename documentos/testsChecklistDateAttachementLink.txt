Test De UI 
Test cases automatizados 
Código existen 3 test de los módulos, agregar fecha, agregar checklist, agregar, adjunto, link, cada test case es un flujo completo , y dentro de los flujos completos se encuentran mas test cases automatizados 
Módulo: Gestión de Fechas

ID: DL-UI-TC-01
Título: Flujo completo creación de fechas 
Descripción: Verificar que se puede agregar una fecha de inicio, fecha de vencimiento, hora y recordatorio en una tarjeta de Trello, y posteriormente eliminarla correctamente.
Tipo de testeo: Automatizado
Prioridad: Alta
Precondiciones:
Usuario autenticado en Trello
Acceso al tablero "Proyecto Final"
Tarjeta "Agregar" disponible
Entorno: El usuario debe tener una computadora de escritorio o laptop con:
Windows 11 pro 23H2 64 bits
RAM 16 GB
CPU Intel i7-1165G7
SSD 1 tera
Navegador Chrome actualizado
Pasos:
Navegar al tablero "Proyecto Final"
Abrir la tarjeta "Agregar"
Hacer clic en el botón "Añadir a la tarjeta"
Seleccionar "Fechas" del menú
Activar el checkbox "Fecha de inicio"
Ingresar fecha de inicio: 16/9/2025
Ingresar fecha de vencimiento: 30/9/2025
Ingresar hora: 2:16
Seleccionar recordatorio: "15 minutos antes"
Hacer clic en "Guardar"
Hacer clic en el badge de fecha para abrir el modal
Hacer clic en "Quitar"
Resultado esperado:
El modal de fechas se abre correctamente
Las fechas se ingresan sin errores
El recordatorio se selecciona correctamente
El badge de fecha aparece en la tarjeta mostrando el rango de fechas
Al hacer clic en "Quitar", la fecha se elimina completamente
El badge de fecha desaparece de la tarjeta
Tipo de prueba: Smoke, Positive, Regression

ID: AE-API-TC-01
Título: Crear una nueva lista en el tablero (POST /lists)
Descripción: Validar que el usuario pueda crear correctamente una nueva lista en el tablero.
Tipo de testeo: Automatizado (API)
Prioridad: Alta
Precondiciones:
•	Usuario autenticado mediante API_KEY y API_TOKEN válidos.
•	Board ID válido disponible en el entorno.
Entorno: Windows 11 Pro, 16 GB RAM, Intel i7-1165G7, SSD 1 TB.
Pasos:
1.	Enviar solicitud POST /1/lists para crear una nueva lista con un nombre válido.
2.	Validar que el esquema JSON de la respuesta cumpla con listSchema.
3.	Confirmar que el campo name coincida con el nombre enviado.
Resultado esperado:
•	La API responde con código 200 OK.
•	La nueva lista se crea exitosamente y su nombre coincide con el solicitado.
•	El esquema de la respuesta es válido.
